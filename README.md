# Calliope Interperter in Python

* How to install
<p>if you want to use this projekt clone this in to your projekts folder using something like git</p>

```
git clone https://github.com/anoraxjhd/{projekt} calliope
```
* For the basic package you will need to import these:
```
import calliope.basic as basic
from calliope.basic import IconNames
```
